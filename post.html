Building fast documentation systems

Agar siz if-elif-else yoki ichma-ich if shartlardan tozaroq va “pythonic” yechim izlayotgan bo'lsangiz, bu aynan siz uchun. Avval Pythonda hammamiz biladigan holatni, ketma-ket shartlarni tekshirishni ko'rib chiqaylik. Tasavur qiling siz foydalanuvchining tizimdagi rolini aniqlaydigan dastur yozyapsiz. An'anaviy usulda bu quyidagicha bo'ladi:

Python funksiyalari haqida to'liq qo'llanma

Python dasturlash tilida funksiyalar --- kodni tartibli, qayta
ishlatiladigan va boshqarilishi oson holga keltiruvchi eng muhim
tushunchalardan biridir.

Kirish

Katta loyihalarda bir xil kodni qayta-qayta yozish katta muammo
tug'diradi.

Masalan:

  • salomlashish
  • hisob-kitob qilish
  • ma'lumot tekshirish
  • fayl bilan ishlash

Bularni har safar qayta yozish o'rniga funksiya yoziladi.

Funksiya nima?

Funksiya --- bu ma'lum bir vazifani bajaruvchi nomlangan kod bloki.

Formula ko'rinishi

Kirish ma’lumotlari → Funksiya → Natija

Hayotiy misol

Choynakka:

  1. suv quyiladi
  2. gaz yoqiladi
  3. qaynatiladi
  4. natija olinadi

Bu ham bir turdagi "funksiya".

Nima uchun funksiyalar kerak?

Afzalliklari

  • Kod takrorlanishini kamaytiradi
  • Loyihani modullarga ajratadi
  • Debug qilishni osonlashtiradi
  • Test yozishni soddalashtiradi
  • Kodni professional ko'rinishga olib keladi

Oddiy funksiya yaratish

Python'da def kalit so'zi ishlatiladi.

def salom_ber():
    print("Salom, dunyo!")

Chaqarish

salom_ber()

Natija

Salom, dunyo!

Parametr va argumentlar

Parametrli funksiya

def salom_ber(ism):
    print(f"Salom, {ism}")

Chaqarish

salom_ber("Ali")
fmt.Println("Ali")
println("Ali")
select

Natija

Salom, Ali

ism --- parametr
"Ali" --- argument

return operatori

Ba'zi funksiyalar natijani qaytaradi.

def qoshish(a, b):
    return a + b

Foydalanish

natija = qoshish(5, 3)
print(natija)

Natija

8

Default parametrlar

Agar foydalanuvchi qiymat bermasa, oldindan tayyor qiymat ishlatiladi.

def salom(ism="Mehmon"):
    print(f"Salom, {ism}")

Misol

salom()
salom("Hasan")

Lambda funksiyalar

Qisqa va bir qatorli funksiyalar uchun ishlatiladi.

kvadrat = lambda x: x * x
print(kvadrat(4))

Natija

16

Muhim eslatmalar

Yaxshi funksiya qoidalari

  1. Nom tushunarli bo'lsin
  2. Bitta vazifa bajarsin
  3. Juda katta bo'lmasin
  4. Keraksiz global o'zgaruvchi ishlatmasin

Yomon misol

def x(a,b,c,d):
    return a+b+c+d

Yaxshi misol

def jami_hisobla(sonlar):
    return sum(sonlar)

Tavsiya

Python'da professional kod yozish uchun:

  • def
  • return
  • *args
  • **kwargs
  • lambda
  • recursive functions
  • decorators

mavzularini ham chuqur o'rganish kerak.
